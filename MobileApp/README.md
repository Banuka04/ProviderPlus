# ProviderPlus Mobile Application

This directory contains the React Native frontend for the ProviderPlus platform, featuring specialized UI flows for the Sinhala-only version, live tracking interfaces, and direct messaging components.

## 🎨 Architecture & Tech Stack
* **Framework:** React Native
* **Build Tool:** Expo
* **Key Integrations:** Real-time mapping for live tracking, websocket connections for direct messaging.

## ⚙️ Local Setup & Installation

**1. Navigate to the mobile app directory:**
```bash
cd mobileapp
```

**2. Install dependencies:**
```bash
npm install
```


## 🚀 Running the App

**Run directly on a device/emulator:**
```bash
# Android
npx expo run:android

# iOS
npx expo run:ios
```

**Start the Expo development server:**
```bash
npx expo start
```

* Press `a` to open in the Android Emulator.
* Press `i` to open in the iOS Simulator.
* Scan the generated QR code with the Expo Go app for physical device testing.
